# 非对称繁简字对照表

2006年10月3日  
发布:振之

此表来自网络，此表主要说明汉字繁简转换并不简单，包括word转换工具有时候也会出错，汉字中有些字在某些义项上繁简异形，存在简体转繁体的问题；而有些义项繁简同形，下列字加“○”以后的字的义项表示它的繁体在作该义项使用时与简化字同形，不变化。如老板的繁体是“老闆”，而木板的繁体还是“木板”，不能写成“木闆”。 

沈如泉2010-11-7
