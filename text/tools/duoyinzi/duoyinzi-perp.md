# 多音字表

## Prepare 多音字表 list text file

```sh
$ perl -ne 'print if /^\d/' duoyinzi-dict.md | wc -l 
108

perl -ne 'if (/^\d/) { s/^\d+(?|?)//; s/(?| *:).*$//; print }' duoyinzi-dict.md | tee ../../info/duoyinzi/duoyinzi-dict-F.txt | ccjf fj -i | tee ../../info/duoyinzi/duoyinzi-dict-J.txt
```
