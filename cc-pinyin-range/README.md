# cc-pinyin-range
Chinese-Character to Pinyin lookup table by range. 
中文转拼音golang实现

## example

```go
package main

import (
    "fmt"
    pybr "github.com/go-cc/cc-table/cc-pinyin-range"
)

func main() {
    s := "天气不错"

    py := pybr.New()
    p, _ := py.Convert(s)

    fmt.Println(p)
    //Tian Qi Bu Cuo


    //设置分隔符
    //首字母是否大写
    py.Split = "-"
    py.Upper = false
    p, _ = py.Convert(s)
    fmt.Println(p)
    //tian-qi-bu-cuo
}
```
